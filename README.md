# devtools.cl

Colección de herramientas online para desarrolladores.  
Rápidas, gratuitas y enfocadas en privacidad: todo funciona directamente en el navegador.

## 🚀 Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Deploy en Vercel

## 🧰 Herramientas incluidas

- JSON Formatter & Validator
- Base64 Encoder / Decoder
- JWT Debugger & Generator
- Password Generator
- UUID Generator
- SQL Formatter
- URL Encoder / Decoder
- HTML Beautifier
- CSS Minifier
- JavaScript Runner
- Markdown Preview
- Lorem Ipsum Generator
- Color Converter
- Cron Expression Editor
- Hash Generator
- Regex Tester
- JSON → YAML Converter
- Timestamp Converter
- JSON Diff
- Curl → Fetch Converter
- URL Parser

## 📁 Estructura

- `/tools/[slug]` → páginas individuales de herramientas
- `/categories/[slug]` → agrupación por categorías (SEO)
- `/sitemap.xml` → generado dinámicamente
- `/robots.txt` → configuración de indexación

## 🔍 SEO

El proyecto está optimizado desde el inicio:

- Metadata dinámica por página
- Sitemap automático
- Robots.txt configurado
- Schema JSON-LD:
  - FAQPage
  - BreadcrumbList
  - SoftwareApplication
- Internal linking entre herramientas y categorías

## 🔒 Privacidad

Todas las herramientas funcionan 100% en el navegador.  
Ningún dato es enviado a servidores externos.

## 🛠️ Desarrollo

Instalar dependencias:

npm install

Levantar entorno local:

npm run dev

Build:

npm run build

## 🌐 Deploy

El proyecto está preparado para desplegar en Vercel sin configuración adicional.

## 🎯 Objetivo

Crear una colección de herramientas útiles para desarrolladores, optimizadas para:

- velocidad
- privacidad
- SEO
- experiencia de usuario
